# research-lab-manager-dbms

## Setup:

1. Clone GitHub Files
2. Install MySQL Connector Python Library in Working Directory:	> pip install mysql-connector-python
3. Establish a server connection in MySQL Workbench and open and execute dbms-setup-brian.sql, dbms-setup-girish.sql, insert-data-brian.sql, insert-data-girish.sql in that order
4. Create a file named .env in Working Directory and insert in it "DB\_PASSWORD=\[Password]" but replace \[Password] with your password (without brackets)
5. Run Menu Application: > python menu-application.py
